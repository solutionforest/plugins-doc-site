# 🗺️ Navigation Management

Build dynamic, multi-level navigation systems for your website with powerful menu management tools.

## 🎯 Overview

Navigation is crucial for user experience. Filament CMS provides flexible tools to create and manage menus that adapt to your content structure.

## 🧭 Navigation Concepts

### Navigation Categories
Think of these as different menu locations on your site:
- **Main Menu** - Primary site navigation
- **Footer Menu** - Links in the footer
- **Sidebar Menu** - Secondary navigation
- **Mobile Menu** - Mobile-specific navigation

### Navigation Items
Individual links within each category:
- **Page Links** - Link to CMS pages
- **External Links** - Link to external websites
- **Custom Links** - Link to any URL
- **Dropdown Menus** - Nested menu items

## 🎬 Creating Navigation

### Step 1: Create Navigation Category

![Create menu](https://demo.solutionforest.net/Filament/cms-plugin/create-menu.gif)

1. **Go to "Page Navigation Categories"**
2. **Click "New Category"**
3. **Configure the category**:
   - **Name**: "Main Menu"
   - **Slug**: "main-menu" 
   - **Description**: "Primary site navigation"

### Step 2: Add Navigation Items

1. **Select your category**
2. **Click "Add Navigation Item"**
3. **Choose link type**:
   - **CMS Page**: Link to existing pages
   - **Custom URL**: External or custom links
   - **Dropdown**: Container for sub-items

### Step 3: Organize Menu Structure

Use drag-and-drop to:
- ✅ **Reorder items** horizontally
- ✅ **Create nested menus** by dragging items under others
- ✅ **Build dropdown menus** with sub-items

## 🔧 Navigation Configuration

### Retrieving Navigation in Code

```php
// Get all navigations
$navigations = \SolutionForest\FilamentCms\Facades\FilamentCms::getNavigations();

// Get specific navigation by slug
$mainMenu = \SolutionForest\FilamentCms\Facades\FilamentCms::getNavigations('main-menu');

// Example usage in Blade template
@foreach($mainMenu as $item)
    <a href="\{\{$item->url\}\}" class="nav-link">
        \{\{$item->title\}\}
    </a>
    
    @if($item->children->count() > 0)
        <ul class="dropdown">
            @foreach($item->children as $child)
                <li><a href="\{\{$child->url\}\}">\{\{$child->title\}\}</a></li>
            @endforeach
        </ul>
    @endif
@endforeach
```

### Navigation in Theme Templates

```php
// In your theme template (e.g., header.blade.php)
@php
    $mainNav = \SolutionForest\FilamentCms\Facades\FilamentCms::getNavigations('main-menu');
@endphp

<nav class="main-navigation">
    @foreach($mainNav as $navItem)
        @include('cms.theme.default.nav-item', ['item' => $navItem])
    @endforeach
</nav>
```

## 🎨 Styling Navigation

### CSS Classes

Navigation items automatically include helpful CSS classes:
```html
<nav class="cms-navigation">
    <a href="/" class="nav-item active">Home</a>
    <div class="nav-item has-children">
        <a href="/about">About</a>
        <ul class="sub-menu">
            <li><a href="/about/team" class="nav-item">Our Team</a></li>
        </ul>
    </div>
</nav>
```

## 📊 Navigation Management Best Practices

### Planning Your Navigation

1. **Map your content** before building menus
2. **Group related pages** logically
3. **Keep main navigation simple** (5-7 items max)
4. **Use descriptive labels** that users understand
5. **Test user flows** through your navigation

### Organization Tips

- **Use consistent naming** conventions
- **Group by user intent** rather than internal structure
- **Consider user mental models** when organizing
- **Provide multiple paths** to important content

### Performance Considerations

- **Limit menu depth** to 3 levels maximum
- **Cache navigation data** for better performance
- **Optimize for mobile** first
- **Test loading times** with complex menus

## 🔄 Navigation Updates

### Automatic Updates

Navigation stays synchronized when:
- **Page titles change** → Menu labels update
- **Page URLs change** → Menu links update  
- **Pages are deleted** → Menu items are removed
- **Pages are unpublished** → Menu items become inactive

### Manual Updates

Manually update navigation when:
- **Reorganizing site structure**
- **Adding new sections**
- **Updating external links**
- **Changing menu labels**

## 🆘 Troubleshooting

### Common Issues

**Navigation not appearing:**
- ✅ Check navigation category exists
- ✅ Verify template includes navigation code
- ✅ Ensure navigation items are added
- ✅ Clear application cache

**Broken menu links:**
- ✅ Verify linked pages are published
- ✅ Check for slug conflicts
- ✅ Update external URLs
- ✅ Test all navigation links

**Styling issues:**
- ✅ Check CSS includes navigation classes
- ✅ Verify theme template structure
- ✅ Test responsive behavior
- ✅ Validate HTML structure

### Performance Issues

**Slow navigation loading:**
- ✅ Implement navigation caching
- ✅ Reduce menu complexity
- ✅ Optimize database queries
- ✅ Use eager loading for relationships
