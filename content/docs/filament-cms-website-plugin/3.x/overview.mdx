## 🚀 Quick Start

> 💡 **New to Filament CMS?** Get started in under 10 minutes!

### 1. **Install via Composer**

```bash
# Add repository
composer config repositories.filament-cms composer https://filament-cms-website-plugin.composer.sh

# Install package
composer require solution-forest/filament-cms-website-plugin
```

### 2. **Setup the Plugin**

```bash
php artisan filament-cms:install
```

### 3. **Register in Panel Provider**

```php
use SolutionForest\FilamentCms\FilamentCmsPlugin;

public function panel(Panel $panel): Panel
{
    return $panel->plugin(FilamentCmsPlugin::make());
}
```

### 4. **Start Creating!** 🎉

Visit your Filament admin panel and start building your website content!

> 📚 **Need detailed instructions?** Check our [Complete Installation Guide](docs/01-installation.md)

Keep your CMS up to date with a single command:

```bash
php artisan filament-cms:update
```

> ⚠️ **Always backup** your database and files before updating!

## 🛠️ Configuration

### Enable Optional Features

```php
// config/filament-cms.php
return [
    'enable_page_tags' => true,     // 🏷️ Page tagging
    'enable_audit_log' => true,     // 📝 Change tracking
];
```

### Publish Resources

```bash
# Customize views
php artisan vendor:publish --tag=filament-cms-views

# Customize translations
php artisan vendor:publish --tag=filament-cms-translations
```
