# Custom Placeholder

Starting from version 2.x, custom placeholders are supported in both templates and campaign content, offering enhanced flexibility and personalization.

Each custom placeholder is identified by a unique key, which can be used in your content like this: `{{custom-placeholder-key}}`.

There are two types of custom placeholders: **Custom Content** and **User Meta**:
- **Custom Content**: Allows you to define and input specific content directly.
- **User Meta**: Lets you specify a user meta key. The application will retrieve the corresponding value from the subscriber's metadata. To update a subscriber's metadata, navigate to the subscriber's edit page and modify the meta section.

This feature empowers you to create more dynamic and tailored content for your audience.

## Minimum Requirement
**version 2.0.0**
